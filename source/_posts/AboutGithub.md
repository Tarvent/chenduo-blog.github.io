# github 基础

## 执行git

```js
git init
```
## github 仓库链接
``` js
git remote add origin [https ]
```
## 拉到本地防止冲突
```js
git pull origin master(main)
```
## 提交本地缓存
```js
git add .
```
## 形成本地仓库的项目文件分支结构
```js
git commit -m 'first commit'
```
## 本地仓库内容提交到github仓库
```js
git push -u origin master
```
